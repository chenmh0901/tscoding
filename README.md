- 前置++和后置++的区别
- 都会改变变量的值
- nums[j]++ is the same as nums[j] = nums[j] + 1
- first return nums[j] then do nums[j]++
- ++nums[j] is the same as nums[j] = nums[j] + 1 then return nums[j]
